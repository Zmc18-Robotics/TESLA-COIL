Here the components that I use :
- Step up 1000Kv module x1
- Eterna cable 1.5mm ( ± 1 meter )
- Copper wire 0.4mm ( ± 75 meters / 350 N )
- 18650 Battery x2
- Aluminium strong wire ( here I use Aluminium Hangers x2 )
- 5⌀ ( 2.5r ) pipe ( 20cm Height )
- Cardboard
- Switch x1
- Crustin terminal x3
- 18650 Battery case x2
